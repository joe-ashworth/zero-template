Quarter Column

This column is an aside that is defined by a class that floats left and has a width of 25%. It can be used inside any size container.

Three simple columns can fill the footer by creating three divs and assigning them the class of col-one-third that floats left and has a width of 33% each. Try to adjust vertical content to the same number of text lines if at all possible for better appearance.

The content in these divs should fill the div as closely as possible. Contact info that is left justified will make the div seem off balance if there is no content on the right side. The p tag padding handles the gaps. Target the footer p to adjust.

If using one of these divs for contact info, consider right floating a map image or simple logo on the left to keep content wide. This last div has a 1% leftover since each is only 33%. Consider adding a class to this div that changes width to 34%.
